六大基础控件

Canvas对象上依附的：

Canvas：画布组件，主要用于渲染UI控件的 Canvas Scaler：画布分辨率只是要组件，主要用于分辨率自适应 Graphic Ray caster：射线时间交互组件，主要用于控制射线相应相关 RectTransform：UI对象位置锚点控制组件，主要用于 控制位置和对齐方式

EventSystem对象上面依附的

Event System和Stand aloneInput Modle：玩家输入事件响应系统和独立输入模块组件，主要用于监听玩家操作

Canvas组件

主要用来负责渲染自己的多有UI子对象

关于Canvas的覆盖模式和摄像机模式：

摄像机模式下 Plane Distance 来控制摄像机和画布之间的距离，Sorting Layer来控制摄像机模式下的摄像机层级问题，相对比覆盖模式，摄像机模式可以将3D物体放到UI之前，

实现类似人物展示界面：摄像机模式用来渲染整个界面（包括3D物体）需要注意的是摄像机模式需要新开一个摄像机不可以用主摄像机，将摄象机模式的相机改为Depth Only保证只渲染UI层面，主摄像机剔除掉UI层面，可保证主摄象机中的3D物体在最后面，因为摄像机模式的相机是Depth Only其他没有物品的地方为透明的，3D物品直接在UI上面渲染
