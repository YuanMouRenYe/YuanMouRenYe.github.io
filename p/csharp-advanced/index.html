知识点六、事件与委托的区别

事件不能在类外部直接赋值，但是可以加减其中的函数

事件不能在类外部直接调用，但是通过在类中封装好调用的函数

事件只能作为成员变量来存在于类中，不能作为临时变量

知识点七、匿名函数的配合使用脱离委托和事件是不会使用匿名函数的

匿名函数的基本语法：

匿名函数可以作为函数的参数也可以作为返回值，同时，匿名函数也不可以直接从事件中-=，因为没有注册的具体名字，无法单独移除，只能直接清除整个事件，直接把整个事件赋值为null，C#：Lambda 和匿名方法功能几乎一样，Lambda 更强（支持异步、表达式树）

在使用匿名函数或者lambda表达式时会产生闭包

闭包的含义：内层的函数可以引用包含在他外层的函数的变量，即使是外层函数的执行已经终止也不会被回收

栈

知识点一、Stack的本质

栈是一种C#封装好的一种数据结构，本质是object[]数组只是封装了特殊的存储规则·

栈是以底层容器完成其所有的工作，对外提供统一的接口，底层容器是可插拔的（也就是说我们可以控制使用哪种容器来实现栈的功能）。

先存入的数据后获取，后存入的数据先获取（遵循先入后出的原则）

知识点二、Stack的申明

需要引用using System.Collections

知识点三、增取查改

增（Push）

压栈(只能一个一个放)

1

stack.Push(1);

取（Pop）

栈中不存在删除的概念，只有弹栈，直接弹出顶部的

1

stack.Pop();

查（Peek,Contains）

无法查看指定位置的元素，只能查看栈顶的内容（没有提供相应的索引 器）（Peek）

查看元素是否存在于栈中（Contains）

改（Clear）

无法直接更改某一个指定元素，所以只能清空所有元素进行压栈和弹栈。

知识点四、遍历

这里提出一个问题他既然都可以找出来他的数量了为什么不能通过索引访问用for来遍历

栈是后进先出的受限线性结构，标准定义只允许在栈顶进行 Push/Pop/Peek 操作，C# 的 Stack类也通过语法封装强制了这一规则，不提供索引器，目的是保证栈的使用规范。所以不能用for 循环遍历

  • 可以遍历长度（Count）

  • 可以用foreach来遍历（遍历出来的顺序是从栈顶到栈底）

  • 也可以强制转换为object数组在进行遍历，顺序也是从栈顶到栈底，（stack.ToArray（））

    到这里就可以用for遍历了

迭代器

迭代器（iterator）有时又称为光标（cursor）是程序设计的软件设计模式 迭代器模式提供一个方法顺序访问一个聚合对象中的各个元素而不暴露其内部的标识

从表现效果上来看是可以在容器对象上遍历访问的接口设计人员无需关系容器对象的内存分配的实现细节 可以用foreach便利的类 都是实现了迭代器的

主要有两种方式：

  1. 传统方式：通过继承两个接口，实现四个方法
  2. 用语法糖：yield return 去返回内容，只继承一个接口

知识点一、标准迭代器的实现方法

  1. 关键接口：IEnumerator,IEnumerable 命名空间：using System.Collections; 可以通过同时继承IEnumerable 和IEnumerator实现其中的方法
  2. 如果想要在自己类中实现迭代器：需要一个名字为GetEnumerator返回值是IEnumerator的方法
  3. 继承接口只是为了便于书写
  4. foraech的本质：
    1. 获取in后面对象的IEnumerator，调用GetEnumerator方法来获取
    2. 执行IEnumerator对象中的MoveNext方法
    3. 只要得到了MoveNext返回值是true就回去得到Current，然后赋值到item
    4. 在完成了一轮之后，光标节点需要重置调用Reset函数一般写在获取GetEnumerator函数中用于第一次重置光标位置

class CustomList:IEnumerable,IEnumerator
{
    private int[] list;
    private int index=-1;

    public CustomList()
    {
        list = new int[] { 1,2,3,4,5,6,7,8 };
    }

    public object Current
    {
        get { return list[index]; } 
    }

    public IEnumerator GetEnumerator()
    {
        Reset();
        return this;
    }

    public bool MoveNext()
    {
        index++;
        return index < list.Length;
    }

    public void Reset()
    {
        index = -1;
    }
}
